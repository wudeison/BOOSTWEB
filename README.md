## Instrucciones para conectar con la base de datos

La configuración de la base de datos se encuentra en `backend/database.js`.

```js
host: "localhost",
user: "root",
password: "tarrao",        // la contraseña de mi MySQL
database: "boost",
```

### Pasos para conectar:
1. Instala MySQL y crea una base de datos llamada `boost`.
2. Usa el usuario `root` y la contraseña `tarrao` para conectarte.
3. Asegúrate de que el servidor MySQL esté corriendo en `localhost`.
4. Si necesitas cambiar la configuración, edita el archivo `backend/database.js`.
